# GitLab Community Edition (Docker Image)

## Usage
1. Add line to /etc/hosts:
```
127.0.0.1 gitlab.vikashkothary.com
```
2. Run the commands in terminal:
```bash
$ cd examples/
$ docker-compose up -d
$ open http://gitlab.vikashkothary.com:
```
